What is Android

Android is a mobile operating system.
Based on a modified version of the Linux kernel and other open source software.
Primarily for touchscreen mobile devices such as smartphones and tablets.

Android Architecture

Why is it secure?

1. Android OS is multiuser Linux system in which each app is defferent user.
2. By Default ,each app gets a unique linux user ID.
3. Each process has its own virtual machine so an app's code run in isolation from other apps.

Android App Components

Building Blocks of app.
Each component is entry point of app.

4 types of activities are:

  1. Activities

    Single Screen with user interface.
    Ex - Email app with activity that shows list of emails.

  2. Services

    Component running background without need to interactwith user.
    Ex - Music playing in background while you chat on whatsapp.

  3. Broadcast Receivers

    Simply respond to broadcast messages from other applications or from the system itself.
    Ex - Applications can also initiate broadcasts to let other applications know that some data has been downloaded to the device and is available for them to use

  4. Content Providers

    Share data between applications.
    Ex- Android System provides content provider that manages user's contact information.

Manifest File -

Declares components ( And we already know that components are building blocks of android app)

Before android system can start app component , system must know component exists by reading app's manifest file


Other Functions


  1. Defines user permissions

  2. Declares min level API level required by application

  3. Declares Hardware and Software features required by application

Project Structure in Android-
