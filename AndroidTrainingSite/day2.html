Overview

Android Layout

Defines structure for a user interface in your app,such as in an activity
All elements in layout are built using hierarchy of view and view group objects.
View : Draws something user can see.
ViewGroup : Invisible container that defines layout structure for view.
Read further here : Common Layouts

Android UI controls

Interactive controls in your app's user interface.

Events

Useful way to collect data about user's interaction with interactive components of application.
3 concepts related to events are-

  1. Event Listeners

    Which listens to event

  2. Event Listeners Registration

    Event Handlers get registerd with event listeners (beech ki kaddi)

  3. Event Handlers

    Handles Event.
