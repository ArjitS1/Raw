2.The Match Parent and Wrap Content Value of Width and Height
Wrap-Content only takes space in which content is present while Match Parent takes whatever width or height parent was having.


3.Aligning Views(widgets) with respect to each other

We can align widgets according to

  • Aligning Edges
    • android:Layout_alignTop="@id/x" - Replace x with that widget's ID ,whose edge we want to be similar to present widget
    • android:Layout_alignBottom="@id/x"
    • android:Layout_alignLeft="@id/x"
    • android:Layout_alignRight="@id/x"
  • Direction from each other
    • android:Layout_below="@id/x" - Replace x with that widget's ID ,whose below we want present widget to be.
    • android:Layout_above="@id/x"
    • android:Layout_toRightOf="@id/x"
    • android:Layout_toLeftOf="@id/x"
